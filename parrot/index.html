🦜 Parrot Assistant v1

A shoulder-mounted, 3D-printed, AI-observant African Grey. Apr 2026

A WiFi-tethered ESP32-S3 wearable that hears, sees, and occasionally decides to comment on what's happening around you. Built in ~2 weeks by an autonomous agent pipeline — BOM, CAD, firmware, Mac-side brain, ordering, and payment tracking all handled agent-side. Only 4 user messages from kickoff to hardware-on-desk.

Hardware

  • Compute: Seeed XIAO ESP32-S3 Sense (onboard OV2640 camera + PDM mic)
  • Audio out: MAX98357A I²S amp + 28 mm 8 Ω speaker
  • Power: external USB-C power bank worn in pocket — no LiPo / no charge circuit
  • Shell: 3D-printed two-part body, magnet-latched, shirt-seam brooch pin mount
  • Total BOM: ~590 PLN delivered-to-doorstep (parts + shipping + print)

Software

  • On-device firmware: PlatformIO + Arduino ESP32 core v3.x. Streams 16 kHz PCM audio + 320×240 JPEGs over WebSocket to the Mac brain.
  • Mac brain: Python asyncio WebSocket server. Reactive loop (Claude Opus 4.7 + ElevenLabs TTS + whisper.cpp STT) + proactive loop ("should I speak?" every 5 min, 20-min cooldown).
  • Daily compute: ~$2.70–4.60/day in API calls at default 5-min proactive interval.
